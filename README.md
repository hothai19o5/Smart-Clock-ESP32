Đây là 1 project tôi lấy ý tưởng từ 1 sản phẩm trên Shopee như hình dưới.

**Bổ sung** thêm 1 số tính năng giải trí:
- Hiển thị biểu cảm robot.
- Chạm để đổi mode.
- Game tích công đức.
